# L’Archipel des Flèches

Un jeu pédagogique original en français pour découvrir progressivement la théorie des catégories.

## Le parcours

Le jeu propose 8 îles et 24 missions :

1. objets et morphismes ;
2. composition ;
3. morphismes identité ;
4. associativité ;
5. diagrammes commutatifs ;
6. foncteurs ;
7. transformations naturelles ;
8. adjonctions.

Chaque notion est introduite par un court rappel, un diagramme et trois questions. Les îles se débloquent progressivement.

Le menu principal propose 10 emplacements de sauvegarde indépendants. Chaque partie est autosauvegardée dans le navigateur et peut être reprise, écrasée après avertissement ou supprimée avec confirmation.

## Jouer

Ouvrez simplement [`index.html`](index.html) dans un navigateur moderne. Le jeu est entièrement autonome : aucune installation, aucun serveur et aucune dépendance externe ne sont nécessaires.

## Publication

Le dépôt est prêt pour GitHub Pages : la branche `main` peut être publiée directement depuis la racine du dépôt.
